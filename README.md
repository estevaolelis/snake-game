# snake-game
Trabalho de faculdade para desenvolver o snake game utilizando Python e logica de Programação Orientada a Objeto

# Ideais
- Precisa separar o corpo da cabeça da cobra
- Ter alimentos diferentes da maça. Ter alimento que dá dano e alimento que aumente a velocidade
- Troca de fase -> Troca de cor do cenario e trocar os alimentos
- Mexer no mapa para ser xadrez - FEITO
- Jogar o score perto do titulo - FEITO
